# Operation

## Task 1:

### Compile app

> `javac -sourcepath src -d classes src/*.java`

### Run server

> `java -cp server.jar task01.Main`

### Run app

> `java -cp classes task01.App localhost 5000`

## Task 2:

### Compile app

> `javac -sourcepath src -d classes src/*.java`

### Run app

> `java -cp classes task02.App [folder name]`
